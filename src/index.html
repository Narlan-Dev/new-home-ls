O que é a residência

A Juventude Dev é uma residência em software que abre as portas da tecnologia para quem nunca programou. Aqui você aprende a pensar como uma pessoa desenvolvedora — e a usar a inteligência artificial como aliada, não como atalho.

Começamos pelo raciocínio: lógica, decomposição de problemas e os fundamentos que sustentam toda a computação. Depois vêm a linguagem, o código e um projeto real que você leva no portfólio.

É um programa do Recôncavo, para o Recôncavo — pensado para jovens da região, de qualquer escolaridade, com prioridade para quem vem da escola pública.
